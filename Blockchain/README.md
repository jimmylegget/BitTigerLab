## 如何学

如何从入门到精通？分为五步：
- 入门篇：掌握区块链相关的基本概念
- 基础篇：能够基于以太坊完成基础智能合约的实现
- 高级篇：能够基于以太坊完成高级智能合约的实现
- 精英篇：能够基于以太坊完成工业级的安全智能合约的实现
- 扩展篇：能够将以太坊智能合约与其它技术结合实现完整的项目

不同需求的学习路径
- 如果你是小白：可以从入门篇开始
- 如果你对区块链已经有所了解：可以从基础篇或者高级篇开始
- 如果你想快速掌握最前沿的区块链应用：可以从精英篇开始
